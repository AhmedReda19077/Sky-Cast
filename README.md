# 🌦️ Weather App

A modern and responsive Weather Application built using **HTML, CSS, and JavaScript**.
It fetches real-time weather data from an API and displays key weather information for any city in a clean glassmorphism UI.

---

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature in Celsius
* 💧 Shows humidity level
* 🌬️ Shows wind speed
* ❌ Error handling for invalid city names
* 🎨 Modern glassmorphism UI design
* 📱 Fully responsive design for all devices
